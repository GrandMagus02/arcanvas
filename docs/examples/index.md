# Examples

Learn how to use Arcanvas through practical examples. Each example includes complete, runnable code.

## Getting Started

- [Basic Canvas Setup](./getting-started) - Initialize Arcanvas and render your first frame

## 2D Graphics

- [Basic Shapes](./basic-shapes) - Draw rectangles, polygons, ellipses, and SVG paths
- [Grid](./grid) - Create an infinite 2D grid with adaptive spacing
- [Selection](./selection) - Interactive object selection with handles
- [Selection on Grid](./selection-grid) - Grid + selectable shapes; tests adorner (handles, getAdornerMeshes)

## Typography

- [Text Rendering](./typography) - Render text with triangulated fonts

## Running Examples

All examples assume you have Arcanvas installed:

```bash
bun add @arcanvas/core @arcanvas/graphics @arcanvas/scene
```

Or use the convenience package for 2D:

```bash
bun add @arcanvas/engine-2d
```

## applyMixins

Applies mixins to a derived class by copying prototype methods from base classes.

| Function | Type |
| ---------- | ---------- |
| `applyMixins` | `(derivedCtor: any, baseCtors: any[]) => void` |

Parameters:

* `derivedCtor`: The derived class constructor.
* `baseCtors`: Array of base class constructors (mixins) to apply.

## createQuadMesh

Creates a full-screen quad mesh for layer rendering.
The quad covers the entire document size with UV coordinates from 0,0 to 1,1.

| Function | Type |
| ---------- | ---------- |
| `createQuadMesh` | `(width: number, height: number) => Mesh` |

Parameters:

* `width`: Document width.
* `height`: Document height.


Returns:

A mesh representing a full-screen quad.

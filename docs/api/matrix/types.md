| Type | Type |
| ---------- | ---------- |
| `AnyMatrix2x2` | `Float32Matrix2x2 or Float64Matrix2x2 or Int16Matrix2x2 or Uint16Matrix2x2 or Uint8Matrix2x2` |

| Type | Type |
